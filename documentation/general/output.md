# Output

This field type returns the email address provided.

**Examples:**

### `mailto`

Return an HTML mailto link.

```
// Twig Usage
{{ entry.example.mailto }}

// API Usage
$entry->example->mailto();
```

### `obfuscated`

Return the obfuscated email.

```
// Twig Usage
{{ entry.example.obfuscated }}

// API Usage
$entry->example->obfuscated();
```
